# .agents Layout

This folder stores local agent assets used by coding assistants in this workspace

## Structure
- `.agents/skills/` reusable task skills with `SKILL.md`
- `.agents/tools/` tool usage guides and guardrails

## Usage
- Keep skills focused and task specific
- Keep tools docs focused on command safety and workflow
- Link every skill and tool doc from `agentic-docs/DIRECTORY.md`
